# Skill Index

This file is checked in as a static reference. Regenerate with `python3 scripts/gen_skills_index.py`.

Use `omni list skills` to get an always-fresh view directly from the filesystem.

## Orchestration & autonomy
- `autopilot` — Full autonomous execution from idea to working code.
- `ralph` — Self-referential loop until task completion with configurable reviewer.
- `ultrawork` — Parallel execution engine for high-throughput task completion.
- `team` — N coordinated agents on a shared task list.
- `ralplan` — Consensus planning gate before execution.

## Planning & exploration
- `plan` — Strategic planning with optional interview.
- `deep-interview` — Socratic crystallization of vague requests.
- `deep-dive` — Trace + deep-interview 2-stage pipeline.
- `deepinit` — Deep codebase init with hierarchical AGENTS.md.
- `external-context` — Parallel doc-specialist web/doc lookup.

## Verification & quality
- `verify` — Prove a change really works.
- `ultraqa` — Test → verify → fix → repeat cycling.
- `debug` — Diagnose session / repo state.
- `trace` — Evidence-driven causal tracing.
- `ai-slop-cleaner` — Regression-safe AI-code cleanup.

## Knowledge & memory
- `remember` — Curate reusable project knowledge.
- `wiki` — Persistent markdown knowledge base.

## Skill & session management
- `skill` — Manage local skills (list / add / remove / search).
- `skillify` — Draft a new skill from the current workflow.
- `cancel` — Cancel any active mode.
- `release` — Generic release assistant.

## Setup & configuration
- `setup` — Install/update routing.

- `omni-doctor` — Diagnose plugin installation.
