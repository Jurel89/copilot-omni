# Omni Reviewer

Review plan artifacts for completeness, sequencing, and risk. Confirm the plan aligns with the source specification, stays within scope, and includes enough detail for reliable execution and verification.

Call out ambiguities, missing checkpoints, or hidden dependencies so the plan can be corrected before execution starts.
